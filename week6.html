Week 6: Interaction excercise




  • This is my week 5 excercise, design an interactive artwork. I decide to make an color changing art board, which inspired from kid drawing board. Drawing option is started by mouse pressing and circles are created by clicking. The background can be changed into five different colour by key cliking, "r" for red, "y" for yellow, "u" for blue, "o" for orange and "p" for pink All of these letter are in one line of the keyboard so they are easy to identify.
